About

Kangoroo is a web application which predicts whether tomorrow will rain or not in Australia. This application contain a machine learning model in backend for predicts the rain in tomorrow. prediction can be done by specifying the location as an input.The ML model was trained and tested by using kaggle dataset which include rainy data in different locations in Australia. The codes for this project can be found in my github repo .More than 80% of Australia has an annual rainfall of less than 600 mm which is less among the all continents other than Antartica which recieves less rainfall.
